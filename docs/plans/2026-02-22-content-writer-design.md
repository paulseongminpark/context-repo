# Content Writer 설계 (2026-02-22)

## 목표
"글을 작성해봐" 등 글쓰기 요청 시 자동 호출되는 전문 글쓰기 에이전트+스킬 시스템.

## 구조
- Agent: `content-writer` [Opus, PROACTIVELY]
- Skill: `/write` (에이전트가 참조하는 프로세스 문서)

## 에이전트 동작
1. 질문 5개 순차 수집 (유형/독자/메시지/톤/길이)
2. 기존 콘텐츠 읽어 스타일 분석
3. 아웃라인 제안 → 승인
4. 초안 작성
5. 퇴고 체크리스트 실행

## 트리거 키워드
글, 써줘, 작성해, 콘텐츠, 포스팅, 아티클, 케이스스터디

## 글 유형
- 포트폴리오 섹션 (AI System, Work, About)
- 기술 블로그 (tech-review)
- 케이스스터디
- 기타 (범용)
